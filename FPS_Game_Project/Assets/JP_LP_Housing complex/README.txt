Description:
--------------------
With this package you can make a "Japanese housing complex".

This package contains:

- 505 meshs. (Polygon count : Min 2 - Max 8691)
- 523 prefabs.
- 7 materials.
- 2 textures. (2048x2048)
- 1 skybox procedural material.
- 1 demo scene.
- 1 post processing stack v2 profile.

"Prefabs/bathtub01_water" sets Water Script and Water Pro Daytime(Unity Standard Assets)

Been used for AssetStore screenshots visualization:
